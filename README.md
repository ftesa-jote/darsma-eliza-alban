# Eliza & Alban Wedding E-Invite

A one-page static wedding invitation for GitHub Pages.

## Publish on GitHub Pages

1. Push these files to your GitHub repository.
2. Go to repository Settings > Pages.
3. Choose `Deploy from a branch`.
4. Select the `main` branch and root folder.

The map button currently uses a placeholder Google Maps query link:
`https://maps.google.com/?q=Restaurant+Ujevara+Prishtine`.
